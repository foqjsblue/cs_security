## ✍️ 작성 형식

모든 항목은 다음과 같은 표 형식으로 정리됩니다:

| 문제 | 키워드 | 요약 |
|------|--------|------|
| ARP 스푸핑이란? | MAC 위조, 감청 | 공격자가 MAC 주소를 속여 패킷을 가로채는 방식 |
| 대칭 vs 비대칭키 | 키 종류, 속도 | 대칭: 하나의 키 / 비대칭: 공개키 + 개인키 사용 |
